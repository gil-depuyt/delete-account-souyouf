Suppression de compte pour l'application Souyouf Academy Brussels

Si vous souhaitez supprimer votre compte et les données associées à notre application Souyouf Academy Brussels, suivez la procédure ci-dessous :

Procédure pour la suppression de compte

Envoyez-nous un email à l'adresse suivante : support@souyoufacademy.com avec les informations suivantes :

  • Votre nom d'utilisateur dans l'application
  • Votre adresse email enregistrée
  • Une confirmation que vous souhaitez supprimer votre compte et toutes vos données

Données supprimées

Les données suivantes seront supprimées après votre demande :

  • Informations de compte (nom d'utilisateur, email)
  • Historique d'utilisation de l'application
  • Tout autre contenu lié à votre compte

Nous nous engageons à supprimer vos données dans un délai de 30 jours suivant la réception de votre demande.

Données conservées

Certaines données peuvent être conservées pour des raisons légales, telles que des obligations fiscales ou réglementaires. Ces données seront conservées pendant la durée nécessaire à ces fins.
